## 项目概览
AlphaHome 是一个基于 Python 的研究与执行框架，聚焦金融数据的采集、处理、因子流水线以及 GUI 驱动的任务编排。仓库包含：
- 数据抓取（Tushare、iFind）与任务化采集
- 因子核心/定义/流水线，支撑量化研究
- GUI 覆盖数据采集、处理、存储配置、任务执行与日志查看
- 处理器（processors）与测试/工具
- 架构、部署、开发与用户文档和示例

## 高层目标
- 提供稳定、可复现、可审计的数据采集能力
- 以模块化流水线支持量化因子研究
- 通过 GUI 降低使用门槛，便于非命令行用户操作数据任务
- 维持清晰的配置、日志与持久化策略

## 关键组件
- alphahome/fetchers：来源适配 API 与批处理器，覆盖股票、基金、期货、指数、宏观、期权、港股、其他等任务定义
- alphahome/factors：因子抽象、因子定义与因子流水线
- alphahome/gui：控制器、处理器（handlers）、服务（services）与各 UI 选项卡
- alphahome/common：配置、日志、数据库管理、模式迁移、规划与任务系统工具
- research/templates：数据库/策略研究模板
- tests/docs/scripts：验证、文档与自动化脚本

## 架构原则
- 数据采集（ingestion）、处理（processing）、呈现（presentation）分层解耦
- 任务驱动执行与显式配置
- 通过模块化包与清晰接口实现可扩展性
- 强化日志与错误处理以提高可靠性
