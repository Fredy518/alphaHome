# AlphaHome 职业转型与开发路线图 (6-Month Plan)

## 核心目标
在6个月内，将 AlphaHome 从单纯的数据平台升级为**"基于基本面逻辑的量化投研系统"**，并产出高质量的实证作品集，完成从 FOF/资产配置向**股票投资/量化基本面 (Quantamental)** 的职业转型。

## 阶段一：因子地基与逻辑代码化 (Month 1-2)
**职业目标**：补齐股票研究基本功，将主观选股逻辑转化为可量化的因子语言。
**战略重心**：**暂时忘记机器学习，专注经典因子的复现与验证。**

### AlphaHome 开发任务
1.  **构建因子库框架 (`alphahome/factors`)**
    *   [ ] 设计统一的因子计算接口 (FactorCalculator)。
    *   [ ] 实现 Barra CNE6 风格的基础因子：
        *   **Size (规模)**: 总市值、流通市值 (对数处理)。
        *   **Value (估值)**: EP, BP, SP, 现金流市值比。
        *   **Momentum (动量)**: RPS (欧奈尔强度), 1个月/3个月/6个月动量, 均线乖离率。
        *   **Quality (质量)**: ROE, ROA, 毛利率, 利润增长率, 盈余稳定性。
        *   **Volatility (波动)**: 日收益率波动, 特质波动率。
    *   [ ] **数据清洗**：处理去极值 (Winsorize)、标准化 (Z-Score)、缺失值填充。

2.  **开发选股扫描器 (`alphahome/analysis/screener.py`)**
    *   [ ] 实现多条件筛选功能 (例如: "ROE > 15% 且 处于250日均线上方")。
    *   [ ] 每日自动生成"符合主观审美"的股票池。

### 职业行动
*   **梳理逻辑**：重新研读欧奈尔、Minervini 等大师著作，提炼出 10-20 个核心指标。
*   **建立观察池**：选定一个基准（如沪深300或中证500），开始每日监控因子表现。

## 阶段二：归因引擎与组合管理 (Month 3-4)
**职业目标**：发挥 FOF 背景优势，构建核心竞争力——**组合归因与风控**。这是你区别于普通研究员的"杀手锏"。
**战略重心**：**构建通用的 `portfolio_analysis` 引擎。**

### AlphaHome 开发任务
1.  **升级归因模块 (`alphahome/portfolio/attribution.py`)**
    *   [ ] 实现通用组合分析逻辑，支持**基于持仓 (Holdings-based)** 的输入。
    *   [ ] 实现 **Brinson 归因**：将超额收益分解为"行业配置收益"与"个股选择收益"。
    *   [ ] 实现 **Barra 风格归因 (青春版)**：计算组合在 Value, Growth, Size 等因子上的暴露度 (Exposure)。
        *   *输出话术："本组合跑赢基准 5%，其中 3% 来自对低估值风格的超配。"*

2.  **开发组合风控面板 (`alphahome/portfolio/risk.py`)**
    *   [ ] 实现风险指标：跟踪误差 (TE), 最大回撤, 个股/行业集中度限制。
    *   [ ] 压力测试工具：模拟"如果发生类似 2015 年股灾，组合会跌多少"。

### 职业行动
*   **逆向工程**：选取 5-10 位你欣赏的公募基金经理，用你的系统分析他们的持仓风格与收益来源。
*   **模拟组合**：在纸面上构建一个模拟组合，每周运行一次归因分析，写周报。

---

## 阶段三：策略闭环与作品集 (Month 5-6)
**职业目标**：产出可展示、可解释、可交易的策略作品，准备面试。
**战略重心**：**闭环验证，可视化展示。**

### AlphaHome 开发任务
1.  **策略回测验证 (`alphahome/backtest`)**
    *   [ ] 开发 2-3 个 **Smart Beta 策略** (例如: "景气度趋势策略" 或 "高质量红利策略")。
    *   [ ] 重点在于**逻辑的白盒化**，而不是收益率的过拟合。
    *   [ ] 完善回测报告生成：包含净值曲线、月度回报热力图、换手率分析。

2.  **可视化与交互 (`alphahome/gui` 或 Web Dashboard)**
    *   [ ] 搭建一个简洁的 Dashboard，整合"市场风格监控"、"选股池"、"组合风控"。
    *   [ ] **这将是你面试时直接演示的 Demo。**

### 职业行动 (求职冲刺)
*   **制作作品包 (Portfolio)**：
    1.  **投资框架白皮书**：阐述你的"主观逻辑+量化风控"体系。
    2.  **策略实证报告**：展示你的 Smart Beta 策略在 A 股的历史表现。
    3.  **归因分析样例**：展示你对某知名基金或市场风格的深度解析。
*   **简历重构**：将 FOF 经历描述为"组合管理与策略研究"，强调数据工程与风控能力。
*   **定向投递**：目标锁定公募基金量化部、券商资管权益部、金工团队、Smart Beta 团队。

---

## 关键原则
1.  **不要黑盒**：任何时候都要能解释清楚为什么买、为什么卖。
2.  **工程化思维**：强调流程的可复制性、数据的准确性、系统的稳定性。
3.  **扬长避短**：不要去拼个股挖掘的深度，要拼组合管理的广度与科学性。
